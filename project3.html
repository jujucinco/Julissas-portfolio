Juju Jams Music Player Web App

As a way to practice my coding skills, I refactored existing Javascript code using JQuery to make a functional music player app. I was also responsible for the app design and branding. The finished app displays album information, song list, and standard music controls for playing, pausing, skipping forward/backward, and adjusting volume.

HTML/CSS Code

Something a little different about coding a project like this is the index.html file was not used for all of the scripting. Here, we only had classes and id's to call in our individual scripts, which contained "dummy data" that functioned similar to an app API. The index.html file was of course still used for the app structure, this is also where I included the background image.

CSS came in handy when I wanted the song number for a play or pause button, as appropriate. I could have used JQuery on the inner HTML of the song number row on hover and click, however DOM insertion is costly on performance. Instead I used CSS classes to hide and unhide elements as appropriate.

Javascript & JQuery

Although I had a solid foundation in Javascript, It was important to me to at least become familiar with JQuery, for it's convenience and efficiency. Although nothing beats how lightweight pure Vanilla JS is, multidisciplinary designers should be familiar with the most common frontend tools at their disposal.

Although I found there was a bit of a learning curve, I liked JQuery for event handlers, such as the on() method. To improve the user experience, I researched how to change the song formatting from seconds to a format users would be more comfortable with, like minutes:seconds. I added a method to the Player class for this purpose called prettyTime and applied it to current times, total times and song durations.

Branding

As this was an educational project for me, instead of a real client, I had as much fun as I could designing and branding. I went with the commonly used dark design pattern for a music app, and tied this with the branding I went with, which was a fun, yet relaxed feel I call "Space Puns". The logo is a combination of a musical note and a J, for the name of Juju Jams. I went for gradient sunrise colors of yellow and pink to pop against the dark starry background image.
